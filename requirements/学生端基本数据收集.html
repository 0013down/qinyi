参数

参数含义

举例

注释

upd_startTime

开始弹奏时间

upd_startTime = 20180208 15:33:45

点击弹奏按钮记录

upd_stopTime

结束弹奏时间

upd_stopTime = 20180208 15:33:45

弹奏完成或中途点击停止按钮记录

upd_duration

持续时间

upd_duration = 00:21:15

upd_duration = upd_stopTime - upd_startTime

upd_songID

曲目ID

upd_songName

曲目名称

upd_songName = "玛丽有只小羔羊"

upd_songOrder

曲目序号

upd_songOrder = 1

该曲目在其书目中的序号

upd_songTempo

曲目标准速度

upd_songTempo = 88

upd_bookID

书目ID

upd_bookName

书目名称

upd_bookName = "小汤1"

upd_mode

弹奏模式

upd_mode = normal

3个参数：normal 普通模式；read 识谱模式；guide 引导模式

upd_hand

分手状态

upd_hand = left

3个参数：left 左手；right 右手；both 双手

upd_startBar

起始小节

upd_startBar = 1

upd_stopBar

结束小节

upd_stopBar = 4

upd_playTempo

弹奏速度

upd_playTempo = 60

if upd_mode != normal { upd_playTempo = -1 }

upd_score

得分

upd_score = 98

if upd_mode != normal { upd_score = -1 }

upd_miss

少弹错误

upd_miss = 5

if upd_mode != normal { upd_miss = -1 }

upd_extra

多弹错误

upd_extra = 4

if upd_mode != normal { upd_extra = -1 }

upd_intonation

音准错误

upd_intonation = 3

if upd_mode != normal { upd_intonation = -1 }

upd_rhythm

节奏错误

upd_rhythm = 1

if upd_mode != normal { upd_rhythm = -1 }

学生端upd（User Play Data 用户弹奏数据）基础数据表

用户每次结束弹奏时记录或上传
